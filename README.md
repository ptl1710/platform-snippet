# Snippets Platform

## Tech
- Next.js (App Router)
- TypeScript
- TailwindCSS
- Prisma (Postgres)
- NextAuth (Credentials/email)
- Vercel for deployment

## Run locally
1. git clone ...
2. copy .env.example -> .env (set DATABASE_URL, NEXTAUTH_SECRET)
3. npm install
4. npx prisma migrate dev --name init
5. npm run dev
